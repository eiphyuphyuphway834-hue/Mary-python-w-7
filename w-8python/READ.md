# Week 8 - Tutorial 8: Food Delivery System

## Description
A simple backend receipt generator for a Food Delivery System, built as a Python module project.

## Files
- `main.py` - Entry point of the program
- `customer.py` - Handles customer data input
- `receipt.py` - Generates and prints the food receipt

## How to Run
```bash
python main.py
```

## Sample Flow
1. Program asks for customer name, food item, quantity, and price
2. Delivery charges are calculated based on quantity
3. A formatted receipt is printed with the total amount
